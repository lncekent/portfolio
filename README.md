# Personal Portfolio Website

![Home Page](https://res.cloudinary.com/dxatb3m2q/image/upload/q_auto,f_auto/v1779334605/Screenshot_2026-05-21_113632_uebebe.png)

This project will be showcasing the plantcam application that I made using MIT App Inventor and Webviewer for frontend. PlantCam is a mobile-based application for plant disease diagnosing, detection and smart recommendation using trained AI Model. By using Google Colab, trained AI with 18,000 images/datasets for solanaceous plants(potato, tomato and pepper).

## Features

- Responsive promotional website
- Home, Privacy, Terms of Service and Disclaimer pages
- APK download link
- AI diagnosis feature explanation
- Figma-based UI implementation

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript, React, Tailwind CSS, Framer Motion, Lucide React, Vite
- **Design:** Google Stitch, Figma
- **Tools:** Git, MCP

## Installation & Setup

1. Open Terminal
2. Clone the repository: `git clone [link]`
3. Change directory to the project: `cd plantcam-promotional`
4. Install dependencies: `npm install`
5. Run the project: `npm run dev` > **Note:** The URL server is `http://localhost:5173`
6. **(Optional)** Building for Production: `npm run build`

## Project Structure

```text
src/
    assets/         Images and visual assets
    components/     Reusable UI components
    pages/          Route pages

```
